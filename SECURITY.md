# Security Policy

## Supported versions

This repository is currently maintained on the default branch. Security fixes are applied there first.

## Reporting a vulnerability

If you find a security issue, please do **not** open a public issue.

Instead, report it responsibly by:

1. Opening a private GitHub security advisory (preferred), or
2. Contacting the maintainer through GitHub with:
   - A clear description of the issue
   - Steps to reproduce
   - Potential impact
   - Suggested mitigation (if known)

We will acknowledge reports as quickly as possible and coordinate a fix and disclosure timeline.
